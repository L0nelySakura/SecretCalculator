# SecretCalculator
Калькулятор с секретным меню с графическим интерфейсом на C++
## Технологии
- C++
- [Qt Framework](https://www.qt.io/)
- [Qt Creator](https://www.qt.io/development/tools)

## Использование
### Требования
Установите зависимости, указанные [выше](#Технологии)
### Сборка и запуск

1) Скачайте репозиторий: 
```sh
git clone https://github.com/L0nelySakura/SecretCalculator
```

2) Откройте проект в Qt Creator

3) Нажмите "Запустить" (Ctrl+R)

После запуска появится отдельное окно с калькулятором. Доступны операции с числами до 25 цифр (учитывая целую и дробную части).
При зажатии "=" на 4 секунды и последующем вводе комбинации "123" в течении 5 секунд, открывается отдельное "секретное" окно
